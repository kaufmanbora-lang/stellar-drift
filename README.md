# Stellar Drift

## ПК и телефон

Это одна и та же игра для ПК и телефона. Рейтинг общий, потому что оба устройства подключаются к одному `server.js`.

1. Запусти сервер:

```powershell
node server.js
```

2. На ПК открой:

```text
http://127.0.0.1:8787
```

3. В игре открой `Архив пилота -> Устройства`.

4. На телефоне открой Wi-Fi адрес из этой вкладки, например:

```text
http://192.168.1.10:8787
```

Телефон и ПК должны быть в одной Wi-Fi сети. На телефоне `127.0.0.1` не подойдёт, потому что это сам телефон, а не твой ПК.

## Аккаунт и рейтинг

- вход по почте и паролю;
- ник, рекорды, ачивки и рейтинг сохраняются на сервере;
- если зайти в тот же аккаунт на телефоне, профиль подтянется;
- мировой рейтинг один общий для ПК и телефона;
- рекорды разделены только по сложности: `Лёгкий`, `Сложный`, `Кибер`.

## Для публикации

Локальная ссылка `http://127.0.0.1:8787` работает только на твоём ПК, а `http://192.168...` работает только пока твой ПК включён и телефон в той же Wi-Fi сети.

Чтобы игра работала на телефоне даже когда компьютер выключен, и чтобы рейтинг был общий по всему миру, выложи папку `stellar-drift` на Node.js-хостинг. После публикации появится настоящая ссылка вида:

```text
https://stellar-drift.onrender.com
```

Эту ссылку можно отправлять другу, открывать на телефоне и ПК, а аккаунты/рейтинг будут общими.

### Быстрый вариант через Render

1. Создай репозиторий на GitHub.
2. Загрузи туда содержимое папки `stellar-drift`.
3. На Render выбери `New -> Blueprint`.
4. Подключи репозиторий с игрой.
5. Render прочитает `render.yaml`, запустит `npm start` и выдаст публичную ссылку.

Файл `render.yaml` уже настроен на постоянную папку `/var/data`, чтобы аккаунты и рейтинг не пропадали после перезапуска сервера.

Полезные переменные окружения:

```text
PORT=10000
SCORE_SECRET=твой_секрет
DATA_DIR=/var/data
```

Для публичного релиза используй HTTPS-ссылку, которую выдаст хостинг.
